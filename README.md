# Design Patterns application

## Running the app

Navigate to each of the following folders and run the main.py file to execute the respective applications.

### ReportCreation

``` bash
cd ReportCreation
python main.py
```

### LibaryIntegration

```bash
cd LibraryIntegration
python main.py
```

### CharacterCloning

```bash
cd CharacterCloning
python main.py
```
